⭐Outlook içerisine rule ekleme: böylece gelen mailin content ve başlığına göre ayrı klasörlenmesi sağlanabilir

⭐açtığım her dokümanın önce her zaman viewing modda gelmesini istiyorum belgelerin güvenliği açısından, bunu ayarlayabilir miyim

Yöntem 1: Word Güven Merkezi (Önerilen)
1. File → Options → Trust Center → Trust Center Settings
2. Protected View sekmesine gel
3. Şu seçenekleri işaretle:
✅ Enable Protected View for files originating from the internet
✅ Enable Protected View for files in potentially unsafe locations
✅ Enable Protected View for Outlook attachments
Bu yöntem dışarıdan gelen dosyalar için otomatik koruma sağlar.

Yöntem 2: Tüm Dosyalar İçin Read Mode (Okuma Modu)
1. File → Options → General
2. Start up options bölümünde
"Open e-mail attachments and other uneditable files in reading view" seçeneğini işaretle ✅

Yöntem 3: Belge Bazlı Kısıtlama (Daha Güçlü)
Eğer başkalarının düzenlemesini engellemek istiyorsan:

Review → Restrict Editing
"Allow only this type of editing: No changes (Read only)" seç
"Yes, Start Enforcing Protection" ile şifre ekle
